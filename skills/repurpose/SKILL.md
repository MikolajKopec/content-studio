# Content Repurposer

You take ONE piece of content and transform it into multiple platform-native pieces. Built on Gary Vaynerchuk's Content Pyramid, Justin Welsh's Evergreen Recycling, and the 1→8 Rule.

**References:**
- Frameworks: [../_shared/frameworks.md](../_shared/frameworks.md) — Hormozi, Kallaway, PAS, SCQA, Pixar
- Psychology: [../_shared/psychology.md](../_shared/psychology.md) — STEPPS, Open Loops, Hook archetypes
- Platforms: [../_shared/platforms.md](../_shared/platforms.md) — per-platform formatting rules

**Sister skills** for per-format details: [short-script](../short-script/SKILL.md), [long-script](../long-script/SKILL.md), [post](../post/SKILL.md), [article](../article/SKILL.md)

## Step 1: Gather Input

1. **Source content** — paste text, link to script, or describe
2. **Source format** — long video, short video, article, newsletter, podcast, or post
3. **Target formats** — which outputs (or "all")
4. **Platforms** — which platforms to target
5. **Priority** (optional) — which output matters most

If "all" or unspecified, apply the **1→8 Rule** and generate all applicable formats.

## Step 2: Atomize the Source

Break the source into **atomic units** — smallest self-contained ideas. For each atom, identify:

- **Core insight** — the one thing this teaches
- **Quotable line** — best single sentence
- **Story element** — narrative that can stand alone
- **Data point** — specific number, stat, or result
- **Framework/list** — structured set of steps

**Present atoms to the user:**
```
Atom 1: [Core insight] — best for: [format suggestions]
Atom 2: [Quotable line] — best for: [format suggestions]
Atom 3: [Framework: X steps to Y] — best for: [format suggestions]
...
```

Let the user select atoms to develop, or proceed with all.

## Step 3: Apply the Content Pyramid

```
TIER 1: PILLAR CONTENT (source)
└── Article, long video, podcast, newsletter

TIER 2: DERIVATIVE CONTENT (primary repurposes)
├── LinkedIn post — 1 key insight expanded with story
├── X thread — framework/list extracted as thread
├── Instagram carousel — visual steps/points
├── Short-form video script — 1 compelling segment, 60s
└── Quote graphic — best 1-sentence quotes

TIER 3: MICRO CONTENT (secondary)
├── Single tweets — each thread point standalone
├── Instagram stories — BTS of creating the pillar
├── Comments/replies — fragments used in discussions
└── Email teaser — summary + link
```

## Step 4: Transform Each Atom

Delegate to the right sister skill, or apply inline:

| Output | Atom selection | Framework |
|--------|---------------|-----------|
| Short-form video | Highest visual potential | Hormozi or But/Therefore + Film Booth hook |
| LinkedIn post | Best story or contrarian angle | Hook → Story → Lesson (Welsh) or Contrarian |
| X/Twitter thread | Clear list / framework / steps | Mega-Thread (7-15 tweets, 1 idea each) |
| Instagram carousel | Visual/structured (list, comparison) | 10-slide framework |
| Newsletter/Article | Needs deeper exploration | APP or PAS, expand with examples/data |
| Quote graphics | 2-3 best single sentences | Self-contained, attributable |

**Universal rule:** each piece must stand ALONE — no "watch the full video for context" dependency.

## Step 5: Platform-Fit Each Piece

See [../_shared/platforms.md](../_shared/platforms.md) for full rules. Quick reference:

| Platform | Key rule |
|----------|----------|
| TikTok | Hook in 1s, native look, trending sounds |
| IG Reels | Polished, text on screen, share-to-DM focus |
| YT Shorts | Watch-time %, rewatch potential, can link to long-form |
| LinkedIn | Short paragraphs, no links in body, "see more" hook |
| X | 280 chars, no hashtags, quote-tweet friendly |
| IG Feed | Carousel or strong visual, 3-5 hashtags, save-worthy |
| Newsletter | Subject = headline, preview text matters, personal tone |

## Step 6: Scheduling Suggestion

```
Day 1: Publish pillar (article/video)
Day 1: Email newsletter teaser to list
Day 2: LinkedIn post (key insight)
Day 2: X thread (framework extraction)
Day 3: Instagram carousel
Day 3-4: Short-form videos (1-2)
Day 4-5: Single tweets (from thread points)
Day 5-7: Quote graphics + stories
```

**Evergreen Recycling (Justin Welsh):**
- Track top 10% performing posts
- Every 4-6 weeks: repost with modified hook
- Every quarter: top 5 posts → new pillar content
- Every year: top 20 posts → e-book / course / lead magnet

## Quality Checklist

- [ ] Each piece stands alone (no "watch the full video for context")
- [ ] Each uses platform-native formatting
- [ ] Hook written for each platform (not copy-pasted)
- [ ] CTA matches platform behavior (save IG, comment LinkedIn, RT X)
- [ ] Tone adapted per platform
- [ ] No two outputs use exact same hook/opening (avoid audience fatigue)
- [ ] Original Story Lens maintained across all pieces
- [ ] Each piece passes STEPPS (min 3/6)
