# VIBE-CODING BEZ KODU #7: SIMPLIFY I BATCH

Platform: TikTok/Reels/Shorts (agnostic) | Length: 60s | Type: Educational
Seria: Porady do vibe-codingu dla osób nietechnicznych

## Hook Options

1. **WOW/Proof:** „Claude właśnie się sklonował. Trzy razy."
2. **Scale:** „Sto agentów. Równolegle. Jeden prompt."
3. **Authority:** „Twórca Claude Code używa tego codziennie. Pokażę ci co."

Rekomendacja: **Hook 1** — WOW factor, antropomorfizacja AI ("sklonował się"), budzi ciekawość. 6 słów. Liczba "trzy" jest konkretna — wciąga.

## Script (Hook 1)

[0-3s] „Claude właśnie się sklonował. Trzy razy."
  📷 Shot: Talking head, bliski kadr, lekki uśmiech niedowierzania
  📝 Text: CLAUDE x3
  🎵 Audio: Triple glitch/clone SFX — trzy szybkie blipy
  ⏱️ Pacing: Powoli, pauza po „sklonował", punch na „trzy razy"

[3-10s] „Od tygodnia mam w Claude Code dwie komendy — /simplify i /batch. Zero instalacji, built-in. I to zmienia WSZYSTKO."
  📷 Shot: Screen recording — terminal z wpisanym /simplify
  📝 Text: /simplify + /batch = BUILT-IN
  🎵 Audio: Notification ding na „built-in"
  ⏱️ Pacing: Szybki, energiczny, podkreślenie „wszystko"

[10-20s] „ALE zanim je zobaczysz — wyobraź sobie, że kończysz feature. Kod działa. TYLE że jest bałagan — powtórzenia, martwy kod, dziwne nazwy. Normalnie musiałbyś to sam przeglądać linia po linii."
  📷 Shot: Talking head → wstawka: chaotyczny diff w edytorze, podświetlone duplikaty
  📝 Text: KOD DZIAŁA ≠ KOD JEST DOBRY
  🎵 Audio: Tension build, niski dron
  ⏱️ Pacing: Wolniejszy, budujący frustrację

[20-30s] „WIĘC odpalam /simplify. I Claude spawnuje trzech agentów — równolegle. Jeden sprawdza reuse, drugi jakość, trzeci wydajność. Trzy review naraz. Jak mieć zespół code reviewerów za darmo."
  📷 Shot: Screen recording — /simplify w akcji, trzy równoległe procesy widoczne w terminalu
  📝 Text: 3 AGENTY → REUSE / QUALITY / EFFICIENCY
  🎵 Audio: Triple whoosh — trzy szybkie dźwięki na spawn każdego agenta
  ⏱️ Pacing: Przyspieszający, excited, energia na „zespół za darmo"

[30-40s] „A teraz /batch. Tu się zaczyna science fiction. Mówisz co chcesz — batch robi interview, tworzy plan, i puszcza DZIESIĄTKI worktree agentów. Każdy pracuje na osobnej kopii kodu. Migracja całego projektu w minuty."
  📷 Shot: Screen recording — /batch fan-out, lista worktree agentów rośnie w terminalu
  📝 Text: /BATCH → DZIESIĄTKI AGENTÓW → MINUTY
  🎵 Audio: Accelerating cascade sound, build-up
  ⏱️ Pacing: Eskalujący, wow-moment na „dziesiątki"

[40-47s] „Boris Cherny — facet, który stworzył Claude Code — powiedział wprost: używam obu codziennie. Codziennie."
  📷 Shot: Talking head, bezpośredni kontakt wzrokowy → overlay: cytat Borisa
  📝 Text: BORIS CHERNY: "I use both daily." — 27 Feb 2026
  🎵 Audio: Drop do ciszy przed „codziennie", single bass hit na powtórzeniu
  ⏱️ Pacing: Wolny, autorytatywny, powtórzenie z naciskiem

[47-53s] „/simplify do review po każdym FEATURE. /batch do migracji na skalę. Nie plugin, nie hack — to jest wbudowane."
  📷 Shot: Split screen — /simplify (lewa) vs /batch (prawa), oba w akcji
  📝 Text: FEATURE REVIEW → /simplify | MIGRACJA → /batch
  🎵 Audio: Dual tone, satisfying click
  ⏱️ Pacing: Szybki, podsumowujący, klarowny

[53-57s] „Claude się sklonował. I czeka na ciebie."
  📷 Shot: Close-up, lean-in, lekki uśmiech
  📝 Text: SKLONOWAŁ SIĘ. ✓
  🎵 Audio: Cisza → clone glitch SFX (callback do hooka)
  ⏱️ Pacing: Pauza. Powoli. Loopable z hookiem.

[57-60s] „A w następnym odcinku — Superpowers plugin. Jak zamienić Claude'a w cały team."
  📷 Shot: Talking head + overlay „#8 SUPERPOWERS"
  📝 Text: #8 → SUPERPOWERS PLUGIN
  🎵 Audio: Outro sting
  ⏱️ Pacing: Szybki, open loop

## But/Therefore Test ✓

> „Claude właśnie się sklonował. Trzy razy." (HOOK)
> SETUP → „/simplify i /batch. Built-in. Zmienia wszystko."
> BUT → „Kod działa — ale jest bałagan. Musiałbyś przeglądać linia po linii."
> THEREFORE → „/simplify spawnuje trzech agentów. Reuse, quality, efficiency. Zespół za darmo."
> THEREFORE → „/batch — dziesiątki worktree agentów. Migracja w minuty."
> THEREFORE → „Twórca Claude Code używa obu codziennie."
> THEREFORE → „/simplify do review, /batch do migracji. Wbudowane."
> THEREFORE → „Claude się sklonował. I czeka na ciebie." (PUNCHLINE → loops to HOOK)

## Production Notes

- **Key visual moments:** (1) Trzy równoległe agenty /simplify w terminalu — główny wow-shot, (2) Fan-out /batch — lista agentów rośnie jak kaskada, (3) Cytat Borisa Cherny na ekranie — social proof, (4) Split screen /simplify vs /batch — clarity moment
- **Recommended audio/music mood:** Dark tech beat z glitch elementami. Triple clone SFX w hooku = audio branding odcinka. Accelerating cascade na /batch segment. Cisza strategiczna przed „codziennie" (47s) i na punchline (53s).
- **Loop point:** „Claude się sklonował" (53s) → „Claude właśnie się sklonował" (0s). Punchline jest echem hooka. Naturalny scroll-restart.
- **CTA type:** Level 5 — open loop na Superpowers plugin (#8). Logiczny next step: poznałeś built-in komendy → teraz najlepszy plugin.
- **Screen recordings needed:** (1) /simplify z widocznym spawnem 3 agentów, (2) /batch z fan-out worktree agentów, (3) Split screen obu komend

## Content Lego Being Tested

**WOW-factor reveal (built-in feature)** — struktura „nie wiedziałeś, że to masz" z proof of authority (cytat twórcy). Testujemy też **antropomorfizację** jako hook mechanic — „sklonował się" zamiast „uruchamia równoległe procesy". Jeśli retention >50% w 10s, antropomorfizacja AI w hookach staje się powtarzalnym wzorcem.

## STEPPS Score: 5/6

- ✅ **Social Currency** — widz wie o built-in komendach, o których większość nie słyszała. Insider knowledge.
- ✅ **Trigger** — za każdym razem gdy kończy feature (→ /simplify) lub planuje migrację (→ /batch)
- ✅ **Emotion** — awe (high arousal) — „trzy agenty naraz" + „dziesiątki worktree agentów" = wow
- ❌ **Public** — terminal output nie jest widoczny publicznie dla innych
- ✅ **Practical Value** — dwie konkretne komendy do użycia natychmiast, zero instalacji
- ✅ **Story** — mini-narracja: bałagan w kodzie → /simplify ratuje → /batch skaluje → twórca potwierdza
